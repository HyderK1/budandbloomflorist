# Bud & Bloom Florist Website

Static Bootstrap website for budandbloomflorist.co.uk.

## What is included
- `index.html` with SEO-focused copy for Wandsworth and South West London florist searches
- Responsive Bootstrap 5 layout
- Optimised WebP/JPG images
- LocalBusiness/Florist schema markup
- Open Graph tags for social sharing
- `robots.txt` and `sitemap.xml`

## Before publishing
1. Connect the enquiry form to your website platform, email provider or form service.
2. Add phone number, email address and opening hours if you want them displayed and included in schema.
3. Upload the contents of this folder to the root of `budandbloomflorist.co.uk`.
